## Available Scripts

### `npm run dev`
- Runs `nodemon src/index.ts`

### `npm run build`
- Runs `tsc -p tsconfig.json`

### `npm run start`
- Runs `node dist/index.js`

### `npm run lint:check`
- Runs `eslint --fix .`

### `npm run lint:fix`
- Runs `eslint --fix .`

### `npm run format:check`
- Runs `prettier --check .`

### `npm run format:fix`
- Runs `prettier --write .`
